Remove Restrictions, Suspensions, and Ammunition Use Codes

    a. From the Maintenance dropdown menu on the SAAS homepage, select Surveillance.

    b. The Surveillance Maintenance page displays. Select a storage point from the dropdown SP menu.

    c. All ammunition items stored at the selected SP display in the table and include the DODIC, NSN/PN, lot number, condition code (CC), TS, and warehouse ID.

    d. Search the SP by DODIC, NSN/PN and/or lot number and click the Search icon. All ammunition items fitting your search criteria will display. Clear the search criteria by selecting the icon.

    e. Click the check box in the Select column for an item. The Restrictions, Suspensions, and Ammunition Use sections of the page open. Any restrictions, suspensions, and ammunition use codes will display for the item.

    f. To remove a restriction, suspension, or ammunition use code, click the Trash Can icon beside the code.

    g. The confirmation message Suspension successfully deleted displays on the Surveillance Maintenance page.
