Add an Individual Entity to Your Entity List

Caution: These tasks are available to admins only.

You can add that single entity to your master entity list. As you do when importing a list of multiple entities, you must validate information about the entity once you add it.

To do this:

  1. Choose Create New Entity from the Entities & Categories drop-down list. The Create New Entity page appears.
  2. Provide the following information:
    Option Description
    About This Entity group
    Entity Title
    Enter a new title in this box. (Entities can be known by several names, so you can use a title to represent all variations of the name.)
    Entity ID
    Enter a new, unique entity ID in this box.
    DUNS Number
    Enter the DUNS number for the entity you are adding. If you are unsure what the number is, enter 00-000-0000. You can edit the record once it's uploaded and enter the correct number.
    Experian ID
    Enter the Experian ID for the entity. If you are unsure what the ID number is, enter 000000000 until you can determine what it is. (If your user subscription includes Experian capabilities, you can click Look up to search a list of possible company matches. To validate the Experian ID, click Validate.)
    Web Address
    Enter the entity's address, either a URL or URI.
    Categories
    To assign an existing category to the entity, choose it from the Assign to drop-down list. To assign a new category, enter a category name in the Create New Category box and then click Add. To remove an existing category, click the X next to the category name.
    Status
    Choose whether to actively monitor the entity by selecting Active. To stop monitoring the entity, select Inactive.
    Note: When you are actively tracking an entity, stories about the entity will appear in the Dashboard and RSS feeds used to track the entity. When you change the status to Inactive, Nexis Entity Insight will neither track nor retain any news about the entity, and news about the entity will be removed from both the Dashboard and any RSS feeds currently monitoring the entity.
    Search Parameters group
    Entity type
    Select Person and/or Company to search articles and display results where the entity has been identified as either an individual or a company. You can leave these options cleared to search for the entity regardless of indexed classification. (See aboutsearchesfinch_ref.htm for an explanation.)
    Entity Name & Variants
    To add or remove a name to be searched, enter a name or variation of a name in the box and click the Add button to include it as one of the search terms. To remove an existing name, click the X next to the name.
    Include Keywords
    To add or remove a keyword or phrase you want to be used when Nexis Entity Insight searches for news stories, enter the keyword in the box and click Add. To remove an existing keyword, click the X next to the term.
    Exclude Keywords
    To keep words from being used when searching for news stories, enter the keyword in the box and click Add. To remove an existing keyword, click the X next to the term.
    Results list This results list shows the news stories that were found based on the search parameters you've used. Use the Risk Angle/Risk Level drop-down lists to filter your results.
  3. Click Create to save the new entity.
